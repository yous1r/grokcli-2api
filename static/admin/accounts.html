协议注册

临时邮箱（MoeMail / YYDS / GPTMail / CF）+ 过盾服务自动建号并导入账号池。每种邮箱服务使用独立的 Key / 域名 / URL，分库存储，互不覆盖。本地过盾以内联方式运行（无需填写 URL）；也可切换到 YesCaptcha 云端。

必填你的 MoeMail 域名；多个域名时批量注册按 1→2→3… 轮询。
API：maliapi.215.im（文档 vip.215.im/docs）。Key 填 AC-…；域名留空则从公开目录随机，多行则批量轮询。收信走 /v1/messages/next 取验证码。
API：https://mail.chatgpt.org.uk · 文档 /zh/api/。Header X-API-Key: sk-…；域名留空则 generate 随机，或填多行轮询。勿再用 gpt-test。
与 MoeMail 的 Base URL / Key / 域名完全独立，切换不会互相覆盖。
API：api.tempmail.lol · 文档 tempmail.lol/zh/api。免费：POST /v2/inbox/create（无需 Key），收信 GET /v2/inbox?token=…

支持多行代理池；批量注册时按策略轮换。共享账号可填下方用户名/密码。
